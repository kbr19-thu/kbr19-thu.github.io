# Borui Kang Personal Website

This repository contains the source for [Borui Kang's personal homepage](https://kbr19-thu.github.io/), built with Hugo and Hugo Blox.

## Local development

1. Install Hugo extended.
2. Start the local server:

```bash
hugo server
```

3. Open the local preview shown in the terminal.

## Main content locations

- `content/_index.md`: homepage sections
- `content/authors/admin/_index.md`: biography, education, and social links
- `content/publication/`: publications
- `content/post/`: news updates
- `content/event/`: talks and posters
- `content/project/`: projects

## Deployment

The site is configured for GitHub Pages at `https://kbr19-thu.github.io/`.
